# 코드 소개
기본적으로 코드는 구동 코드(content_script.js) / API연결 코드(service_worker.js) / 기타 설정(manifest.json)으로 나누어집니다. 

API에게 요청을 시도하려면 API_key가 있어야 합니다. key가 있다면 service_worker.js에서 입력하세요.
> .js확장자는 연결프로그램 -> 메모장으로도 수정 가능합니다.

### Key 세팅 방법
`google Aistudio`에 접속하여 로그인 후 왼쪽 사이드 패널에서 History 밑에 Dashboard를 엽니다. 그 경로로 key를 받을 수 있습니다. 

#### 주의
2.5-pro는 하루 20회 요청 제한이 있습니다. flash도 비슷할 거에요. 저는 카드 연결해서 쓰긴하는데 1.5-pro, 1.5-flash, 2.0-flash, 2.0-flash-lite 등 다른 모델로 바꿔가면서 해도 코드 **테스트용도**로는 쓸 수 있을겁니다.

# 확장프로그램 실행 방법
크롬 -> 오른쪽 위 설정 -> 확장프로그램 -> 확장 프로그램 관리 -> 오른쪽 위 개발자 모드 실행 -> 왼쪽 위 "압축해제된 확장 프로그램 로드" 클릭 -> 확장프로그램 파일들이 들어있는 폴더 선택 -> 서비스워커 클릭(실행 후 나오는 모든 메세지가 여기에 출려됩니다.)

> 이제 아무 네이버 뉴스로 진입하면 자동으로 서비스워커에서 현재 동작 중인 상태의 메세지가 출력될 것입니다.
