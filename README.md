# 가짜뉴스 판별 웹 애플리케이션

-------------

```
프로젝트 폴더 
├── app.py          - 플라스크 실행 파이썬 코드  
├── static/         - 정적 파일이 저장된 폴더
│   ├── style.css   - 메인 / 결과 페이지의 스타일시트  
│   └── script.js   - 메뉴 버튼 관련 자바스크립트  
└── templates/      - html이 저장된 폴더  
    ├── index.html  - 메인 페이지 html  
    └── result.html - 결과 페이지 html  
```
